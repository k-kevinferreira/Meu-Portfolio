💼 Portfólio Profissional – Desenvolvedor Web

Este projeto consiste no desenvolvimento de um portfólio profissional voltado para apresentação de projetos, tecnologias utilizadas e trajetória no desenvolvimento web.

O objetivo é criar uma presença profissional online, permitindo demonstrar:

habilidades técnicas

organização de código

boas práticas de desenvolvimento

evolução na área de software

O site funciona tanto como apresentação para recrutadores quanto como vitrine de projetos para clientes.
