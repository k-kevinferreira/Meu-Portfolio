# 💼 Portfólio Profissional – Desenvolvedor Web

Este projeto consiste no desenvolvimento de um portfólio profissional voltado para apresentação de projetos, tecnologias utilizadas e trajetória no desenvolvimento web.

O objetivo é criar uma presença profissional online que permita demonstrar habilidades técnicas, organização de código e evolução na área de desenvolvimento de software.

---

# 🧠 Sobre o Projeto

O portfólio foi desenvolvido utilizando tecnologias modernas de front-end com foco em:

- interface limpa e profissional
- experiência do usuário
- navegação simples
- layout responsivo
- organização de componentes

O projeto apresenta:

- perfil profissional
- tecnologias utilizadas
- projetos desenvolvidos
- área de contato

Este site serve tanto como **apresentação profissional para recrutadores** quanto **vitrine de projetos para clientes**.
